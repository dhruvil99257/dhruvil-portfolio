# Dhruvil Sheladiya Portfolio

Static, Vercel-ready portfolio website for QA Analyst and Lead Generation roles.

## Files

- `index.html` — website content
- `styles.css` — complete responsive design
- `script.js` — dark/light mode, mobile menu, contact form mailto
- `assets/Dhruvil_QA_Resume.pdf` — QA resume download
- `assets/Dhruvil_Lead_Generation_Resume.pdf` — Lead Generation resume download
- `vercel.json` — Vercel static deployment config

## How to deploy on Vercel

1. Create a new GitHub repository, for example `dhruvil-portfolio`.
2. Upload all files from this folder to the repository.
3. Go to Vercel and choose **Add New → Project**.
4. Import the GitHub repository.
5. Framework preset: **Other** or **Static**.
6. Click **Deploy**.

## Edit content

Open `index.html` and update text, links, numbers, or section details.
